1. O que é TypeScript e quais são suas principais vantagens em relação ao JavaScript?
TypeScript é um superset (superconjunto) do JavaScript que adiciona tipagem estática e recursos avançados de desenvolvimento.

Vantagens:

Tipagem Estática: Permite detectar erros de tipo em tempo de desenvolvimento, antes da execução do código.

Melhor suporte a IDEs: Ferramentas como o VS Code oferecem autocompletar, navegação entre arquivos, e refatoração inteligente, graças aos tipos.

4. O que é o arquivo tsconfig.json e qual sua importância?
O tsconfig.json é o arquivo de configuração do compilador TypeScript. Ele define como o TypeScript deve compilar o código do projeto.

Importância:

Permite padronizar configurações do projeto.

Evita necessidade de passar parâmetros toda vez que rodar o tsc.

Duas configurações importantes:

"target": Define para qual versão do JavaScript o código será convertido (ex: "ES6").

"outDir": Define o diretório onde os arquivos JavaScript compilados serão salvos.
